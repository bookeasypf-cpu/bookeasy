# 🎯 Mesurer l'impact

Après 30 jours, regarde :

**Google My Business** :
- Nombre d'impressions search (combien de fois ta fiche apparaît)
- Nombre d'actions (clic site, clic téléphone, clic itinéraire)
- Top requêtes qui amènent à ta fiche
- → Disponible dans le dashboard GMB

**WhatsApp Business** :
- Nombre de conversations entrantes
- Messages envoyés / reçus
- Taux de réponse au message d'accueil
- → Disponible dans Settings > Business Tools > Statistics

**Cross-canal** :
- Combien de prospects ont contacté via WhatsApp suite à GMB
- Combien de signups Pro proviennent du link `wa.me` dans la bio Insta
- → À traquer manuellement dans un Google Sheet `marketing/distribution/leads-tracking.csv`
