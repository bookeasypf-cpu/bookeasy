API Génération Auto

/api/marketing/og · génère un visuel marketing depuis un merchant ID

⚡ Photo merchant authentique pour les plans PRO

🔧 Configuration

Choisis un marchand et un template

?merchantId=...&template=hero&format=square

📸 Aperçu généré

L'image est servie par Next.js avec cache 1h

🔄 Comment ça marche en production

Pipeline complète de génération marketing

// 01

Marchand signup

Webhook trigger sur création de merchant + upload de coverImage

// 02

Génération auto

Appel à /api/marketing/og avec merchantId + template selon le plan

// 03

Notification admin

Mara reçoit l'aperçu par mail, valide et publie

// 04

Publication Buffer

Visuel poussé vers Buffer/Hootsuite avec caption du calendrier

💎 Différenciation Plan PRO

Plan FREE → photo Unsplash générique du secteur (anonyme)
Plan PRO → photo réelle du marchand en arrière-plan + son nom + ses services
→ Argument de vente direct dans le pricing : "Tes visuels marketing, générés automatiquement avec ta photo."
