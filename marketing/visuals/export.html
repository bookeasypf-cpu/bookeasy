📖 Comment utiliser ces fichiers

📅 Sur Buffer

  1. Publish → Queue → ⚙ → Bulk Upload
  2. Glisse-dépose le CSV téléchargé
  3. Map les colonnes (auto-détection souvent OK)
  4. Review chaque post avant validation
  5. Buffer programme automatiquement aux dates/heures du CSV

🦉 Sur Hootsuite

  1. Planificateur → Bulk Composer
  2. Upload CSV (format Hootsuite obligatoire)
  3. Sélectionne les comptes sociaux cibles
  4. Programme en 1 clic
  5. ⚠️ Hootsuite ne supporte pas les images via URL — il faut joindre manuellement après upload

💡 Workflow recommandé : Utilise Buffer (gère bien les URLs d'images). Lance le serveur Next.js + l'API /api/marketing/og pour que les URLs preview résolvent vers de vraies images au moment de la programmation.
