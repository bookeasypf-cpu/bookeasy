À faire en parallèle :

  4. Reposter en Reel avec son tendance = portée x10
  5. Tagger les marchands partenaires = effet réseau

📸 Exporter en PNG/JPG pour publier

  1. Ouvre le secteur dans Chrome → Cmd+Shift+4 sur Mac (ou outil de capture Windows)
  2. Sélectionne la zone du visuel → screenshot direct en haute qualité
  3. Ou : clic droit sur le visuel → "Capturer un nœud" dans DevTools Chrome (qualité max)
  4. Format final : PNG 1080×1080 (post) ou 1080×1920 (story) — compatible Instagram, Facebook, TikTok
