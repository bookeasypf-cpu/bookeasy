## 🔑 Sources exploitées

1. **Polynésie Pratique** — annuaire métier tatoueurs/coiffeurs/instituts (50+ entrées tatoueurs)
2. **Rendez-vous.pf** — directory pro beauté/santé (55 coiffeurs, 46 instituts)
3. **CPS.pf** — Caisse de Prévoyance Sociale, liste officielle conventionnés (PDF 60 pages)
4. **OIPF** — Ordre des Infirmiers PF (164 infirmiers libéraux)
5. **Fenua Wellness** — partenaires bien-être Tahiti
6. **Lemedecin.fr / rdvdentiste.net** — annuaires médicaux
7. **Instagram / Facebook** — recherche directe pour tatoueurs/maquilleuses/barbers

## ⚙️ Notes méthodologiques

- **Téléphones** : format international `+689 XX XX XX XX` (XX numéro local)
- **CPS conventionné** : pros remboursés par la Caisse de Prévoyance Sociale = base fiable
- **Vini** : numéro mobile (préfixe 87/88/89 en Polynésie)
- **Duplicates** : certains pros apparaissent dans 2 secteurs (ex: Bastien Coupez = ostéo + kiné CPS) — c'est volontaire pour traçabilité source
- **Emails** : 32 emails directs récoltés, le reste via téléphone/FB DM/formulaire site
